== Description ==

This is instaSHOW, a plugin for displaying your instagram content in a wordpress page!

== Installation ==

1. Upload the folder `dfy_instaSHOW` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Select instaSHOW from the settings menu
4. Enter your username, etc.
5. Place `[instagram]` in any post or page.
6. Done!
